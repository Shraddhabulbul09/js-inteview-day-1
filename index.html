Ans: Both are the comparision operators == is used to comapre the value where === is used to compare both value and datatype.

           // ===

Q2. What are the differences between var, let and const?

Var:- var is a keyword for creating variable. it can be redeclared and reinitialized.It has a global scope

let:- It cannot be redeclared but can be reintialized.It has a block scope.

const:- it cannot be redeclared and cannot be reintialized.It also has a block scope.

Q3.What is hoisting?

Ans:-It is the behavior of javascript to move variables to the top of their scope. It allows us to use variables before its declaration. Let and const cannot be hoisted it will give you reference error.

                console.log(a);

Q6.What are pure functions?

A pure function in JavaScript is a function that returns the same result if the same arguments(input) are passed in the function.

function add(a,b){
            return a+b;
        }
        
        const result = add(1,2)
        console.log(result); //3
